# Imágenes de Proyectos

Esta carpeta contiene las imágenes para cada proyecto del portafolio.

## Estructura

Cada proyecto tiene su propia carpeta con el mismo nombre que el slug del proyecto (nombre del archivo JSON sin la extensión).

## Nombres de archivos

Para cada proyecto, guarda las imágenes con estos nombres:

- **`hero.png`** - Imagen principal del proyecto (1920x800px recomendado)
- **`after-reto.png`** - Imagen después de la sección "El reto" (1200x600px recomendado)
- **`investigacion.png`** - Imagen de la sección de investigación del proceso (1200x600px recomendado)

**Formato estándar**: PNG (exportado desde Figma)

## Ejemplo

```
public/proyectos/
  └── calipso-sistema-diseno/
      ├── hero.png
      ├── after-reto.png
      └── investigacion.png
```

## Dimensiones recomendadas

- **Hero**: 1920x800px (ratio 2.4:1)
- **After Reto / Investigación**: 1200x600px (ratio 2:1)

## Formato estándar: PNG

**Formato utilizado**: PNG (exportado desde Figma)

### Ventajas de PNG:
- ✅ Sin pérdida de calidad
- ✅ Soporta transparencia
- ✅ Ideal para interfaces y mockups
- ✅ Exportación directa desde Figma

### Optimización (opcional):
Si necesitas reducir el tamaño de archivo más adelante, puedes:
- Usar herramientas como TinyPNG o ImageOptim
- Convertir a WebP con herramientas externas (si es necesario)
